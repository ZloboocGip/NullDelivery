NullDelivery - Delivery Missions System for Spigot/Paper

Region-safe delivery missions for Minecraft servers.

NullDelivery is a lightweight delivery plugin where players receive package missions and must deliver them to random locations inside configured worlds.

Supports bossbar timers, cooldowns, safe coordinate generation and custom reward commands.

━━━━━━━━━━━━━━━━━━━━━

FEATURES

✔ Random Delivery Targets
Generate random delivery locations inside configured worlds.

✔ Safe Location System
Avoids WorldGuard regions and nearby protected areas.

✔ BossBar Navigation
Live timer and distance display while delivering.

✔ Reward Commands
Run any console commands after successful delivery.

✔ Cooldown Support
Optional cooldown between deliveries.

✔ World Restrictions
Allow deliveries only in specific worlds or only through console/NPC systems.

✔ Lightweight & Optimized
Built for modern Spigot/Paper servers.

━━━━━━━━━━━━━━━━━━━━━

DELIVERY SYSTEM

Players start a delivery mission and receive a target location.

They must reach the destination before time runs out.

Features:

• random safe coordinates
• configurable delivery radius
• bossbar timer
• distance tracking
• cooldown system
• console reward commands

Perfect for:

• Survival Economy
• RPG servers
• Quest systems
• NPC deliveries
• City roleplay

━━━━━━━━━━━━━━━━━━━━━

COMMANDS

/delivery get
Start a delivery mission.

/delivery info
Show current delivery info.

/delivery reload
Reload plugin configuration.

Aliases:

• /deliv
• /nulldelivery

WHY NULLDELIVERY?

Instead of passive money generation, players travel through the world and complete active delivery missions.

Simple, lightweight, and perfect for economy-based gameplay.
